  # Privacy Policy

  **Claude Navigator** does not collect, store, or transmit any user data.

  ## Data Collection

  This extension only operates locally on the user's browser when visiting claude.ai. It:
  - Reads page content to extract conversation titles for navigation purposes
  - Stores user preferences (highlight duration, auto-hide settings) locally using Chrome's storage API
  - Does not collect, transmit, or share any personal information

  ## Third Parties

  We do not share any user data with third parties.
